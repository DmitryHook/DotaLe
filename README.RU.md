# DotaLe — Hero Guesser


> **Угадай героя Dota 2** по его характеристикам: пол, раса, роль, атрибут, тип атаки, сложность и год выхода.

---

## Как играть

1. Начни вводить имя героя в поле поиска и выбери его из списка.
2. После каждой попытки ячейки подсветятся в зависимости от результата:
   - 🟢 **Зелёный** — совпадает
   - 🟡 **Жёлтый** — частичное совпадение
   - 🔴 **Красный** — не совпадает
3. Если застрял — подсказки открываются автоматически по мере попыток:
   - 💬 **Цитата** — после 4 попыток
   - ⚡ **Способность** — после 8 попыток
   - 🖼 **Экран загрузки** — после 12 попыток
4. Угадай героя за как можно меньшее количество попыток!

## Запуск проекта

### 1. Клонируй репозиторий

```bash
git clone https://github.com/DmitryHook/DotaLe
cd dotale
```

### 2. Создай виртуальное окружение

```bash
python -m venv .venv
```

**cmd:**
```cmd
.venv\Scripts\activate.bat
```

**PowerShell:**
```powershell
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
.venv\Scripts\Activate.ps1
```

**Mac/Linux:**
```bash
source .venv/bin/activate
```

### 3. Установи зависимости

```bash
pip install -r requirements.txt
```

### 4. Примени миграции

```bash
python manage.py migrate
```

### 5. Загрузи данные в базу

```bash
python manage.py fill_db
```

### 6. Запусти сервер

```bash
python manage.py runserver
```

Открой [http://127.0.0.1:8000](http://127.0.0.1:8000) в браузере.
