What is DotaLe?

DotaLe is a free browser game inspired by the famous Wordle. Instead of words — Dota 2 heroes. Each guess reveals clues about the hidden hero. The fewer attempts, the better.

1

Type a Name

Start typing a hero name for autocomplete. All heroes are in the pool.

2

Read the Colors

Green means "match". Yellow means "partial". Red means "wrong". Every cell is a clue.

3

Use the Hints

After 4, 8, and 12 attempts a quote, an ability, and a loading screen are revealed.

4

Guess It!

Your goal is to guess the character as quickly as possible.

Color System

After each guess, 7 hero traits are highlighted by color. Analyze the pattern and narrow down your search.

Match This trait of the guessed hero fully matches the hidden hero.

Partial There are shared values — for example, one of the roles matches.

No Match This trait is completely different from the hidden hero.

FAQ

Do I need to know all Dota 2 heroes?

The better you know Dota, the faster you'll guess. But even a newcomer can logically narrow things down using the color clues. You can also consult the Dota 2 hero encyclopedia.

Can I guess the same hero twice?

No. Already guessed heroes disappear from autocomplete, and the server blocks repeated attempts with the same name.

What does a yellow partial match mean?

Some fields can have multiple values — for example, a hero can fill three roles at once. Yellow means at least one role (or species) matches, but not all of them.

Is the game updated?

The hero database is updated alongside Dota 2 patches. New heroes are added to the pool after their release.
